Código fonte / Pré-formatação

O comando document.getElementById('teste') é escrito em linguagem Javascript.

O comando document.getElementById('teste) é escrito em linguagem Javascript.

Note a diferença na formatação entre o nome dos comandos marcados, no primeiro o comando está difícil de ler, mas no segundo, usamos a tag <code>, que torna o mesmo comando mais legível ao ser escrito.

Abaixo, vejamos um exemplo com um código Phyton

num = int(input('Digite um número')) if num % 2 == o: print(f'O número {num} é PAR') else: print(f'O número {num} é ÍMPAR') print('Fim di programa')

O código acima, já está dentro da tag <code>, mas perdeu a sua indentação, para mantê-lo da forma que ele está dentro do editor de códigos, iremos envelopar toda a tag <code> dentro de uma tag <pre>.

        
            num = int(input('Digite um número'))
            if num % 2 == o:
                print(f'O número {num} é PAR')
            else:
                print(f'O número {num} é ÍMPAR')
            print('Fim di programa')

Caso você queira colocá-lo mais alinhado com o texto dos parágrafos, é só selecionar o código, pressionar shift + tab e arrastá-lo.
