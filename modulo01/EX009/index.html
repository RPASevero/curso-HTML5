Trabalhando com listas em HTML5

As listas simplificam e sintetizam o conteúdo a ser passado e faz a demarcação de cada item para facilitar a visualização.

Listas ordenadas

Para criar uma ordered list, vamos usar a tag <ol> para delimitar a lista e <li> (list item) para identificar cada item da lista.

Na lista abaixo, não foi usado parâmetro específico, então naturalmente, ela puxa a ordenação númerica.

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  5. Item 5

No caso abaixo, foi usado o type="A" dentro da tag <ol>.

  1. Item 1
  2. Item 2
  3. Item 3
  4. Item 4
  5. Item 5

Aqui, foi utilizado o type="a" dentro da tag <ol>.

  1. Item 1

Aqui, o type="I" dentro da tag <ol>.

  1. Item 1

E aqui,o type="i" dentro da tag <ol>.

  1. Item 1

Listas não ordenadas

Para listas não ordenadas, usamos a tag <ul>.

Na lista abaixo, não foi usado parâmetro específico, então naturalmente, ela puxa a ordenação utilizando uma bolinha preenchida, disc.

  • Item 1
  • Item 2
  • Item 3
  • Item 4
  • Item 5
