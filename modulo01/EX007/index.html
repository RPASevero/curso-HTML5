Principais formatações

Negrito / Destaque

Nessa frase, temos um termo em negrito usando a tag <b> que é um exemplo de tag não semântica.

Já nessa outra frase, eu tenho um termo em negrito usando a tag <strong> que é uma tag semântica.

Itálico / Ênfase

Agora, eu tenho um termo em itálico, usando a tag <i>, tag não semântica.

E nesta frase, o meu termo em itálico está utilizando a tag <em>, que é uma tag semântica.

Texto marcado

Exemplo de marcação de texto usando a tag <mark>.

Texto grande e pequeno

Para criar texto grande e texto pequeno, usamos as tags <big> (que já está obsoleta) e <small>, respectivamente.

Texto deletado

Marcamos um texto como excluído para indicar que ele deve ser lido, mas não considerado, a tag utilizada foi a <del>.

Texto inserido

Marcamos um texto inserido para darmos uma ênfase e indicar que ele foi inserido depois, a tag utilizada foi a <ins>.

Texto sobrescrito

Esta tag serve para coisas do tipo x20+3, a tag utilizada foi a <sup> com o caracter a ser sobrescrevido.

Texto subscrito

E a subscrita, para inserir por exemplo H2O, a tag utilizada foi a <sub> com o caracter a ser subscrito.
